these classes will all be located in the game.py file
game:
    This class will contain all data about the game that isn't part of any other class an the functions for intteracting wih the game.
    It will also contain all object that are a part of the game and all data that isn't part of another object. 
player:
    This class contains all information about the player (for example position, speed, direction). It should also contain the player movement code. 
enemy:
    This class should contain the basic code an data about the enemies, for example the movement code (for pathfinding to the targer) and the transitions between modes
blinky/pinky/inky/clyde: 
    These classes inherit from enemy, but add their own specific code. This is the target setting and the ghost house behaviour.
